Restler 1.0.0-beta5

A powerful REST API Client for Android.

Restler allows you send custom HTTP/HTTPS requests and test your REST API anywhere and anytime.

Why Choose Restler?

Restler has been built with simplicity and ease of use in mind.

Easy to use

Specify URL, headers, and authorization all in one place. Then just hit send.

Organize everything

Create folders, save your requests, and import and export your data.

Beautiful Design

Intuitive and beautiful interface using Material Design.

Features & Details

Insomnia integration, Collections, Authentication and more make Restler one of the most powerful REST API clients on Android.

History

Requests you send are automatically saved so that you can come back and review at any time.

Collections

Save and organize your requests in folders and load requests without writing its attributes again.

Favorites

Favorite requests and folders so they can be shown at the top and accessed quickly.

Auto-Complete

Restler autocompletes headers with commonly-used values.

Authentication

Authenticate your requests with Basic, Bearer Token and Hawk Auth.

Insomnia Support

Import and export Insomnia collections.

View Response Details

Parses and displays the complete raw response including headers, cookies, status, etc.

Custom Methods

Supports the most common HTTP methods. Also supports custom methods.

Request Body

Supports to Multipart, Form Url Data, Binary File and Text request body types.

Response Copy

Quickly copy the response body received.

Http & Https

Switch easily between HTTP and HTTPS.

Cookies

Manage cookies and send them with your request.
